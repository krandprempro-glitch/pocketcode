```java
// 统一异常处理格式
public class RemoteFileOperationException extends Exception {
    public enum ErrorType {
        NETWORK_ERROR,
        PERMISSION_DENIED,
        FILE_NOT_FOUND,
        INVALID_PATH
    }
    
    private final ErrorType errorType;
    
    public RemoteFileOperationException(ErrorType type, String message, Throwable cause) {
        super(message, cause);
        this.errorType = type;
    }
    
    public ErrorType getErrorType() {
        return errorType;
    }
}
```

```markdown
## 远程文件浏览器功能

### 功能特点
- 🔗 SSH/SFTP协议连接远程服务器
- 📁 VSCode风格的目录树浏览
- 🔖 文件夹书签收藏功能
- 💾 项目工作区管理
- 📱 移动端优化的用户界面

### 使用方法
1. 点击SSH浮动按钮选择"远程文件浏览"
2. 配置SSH连接参数
3. 连接成功后即可浏览远程文件

### 功能说明
- **目录树**: 左侧抽屉显示完整目录结构，支持展开折叠
- **书签管理**: 收藏常用目录，快速跳转
- **项目切换**: 支持多个服务器项目，独立管理
- **文件操作**: 预览、下载、查看信息等操作

### 技术架构
- 基于SSHJ库实现SSH/SFTP连接
- RxJava处理异步操作
- RecyclerView实现高性能列表
- SharedPreferences存储配置和状态
```

```markdown
# 远程文件浏览器用户手册

## 快速开始

### 连接服务器
1. 在Termux主界面点击SSH浮动按钮
2. 选择"远程文件浏览器"选项
3. 首次使用会显示快速连接面板
4. 填写服务器信息：主机地址、端口、用户名、密码
5. 点击连接按钮

### 浏览文件
- 主界面显示当前目录的文件列表
- 点击文件夹进入子目录
- 点击文件显示操作菜单
- 使用面包屑导航快速返回上级目录

### 使用目录树
- 从左侧划出或点击菜单按钮打开抽屉
- 点击文件夹图标展开或折叠目录
- 点击文件夹名称在主界面中显示该目录
- 展开状态会自动保存

### 管理书签
- 长按任意文件夹添加到书签
- 在左侧抽屉的书签区域快速访问
- 长按书签可以重命名或删除

## 高级功能

### 项目工作区
每个服务器连接会自动创建一个项目工作区，包含：
- 连接配置信息
- 目录展开状态
- 书签收藏列表
- 最后访问位置

### 多服务器管理
- 通过左侧抽屉顶部的下拉菜单切换项目
- 每个项目独立维护自己的状态
- 最近使用的项目会优先显示

### 文件操作
支持的操作包括：
- **查看内容**: 预览文本文件内容
- **下载文件**: 保存到本地存储
- **复制路径**: 复制完整文件路径
- **文件信息**: 查看详细属性

## 故障排除

### 连接问题
- **连接超时**: 检查网络连接和服务器地址
- **认证失败**: 确认用户名密码正确
- **权限不足**: 确保用户有相应目录访问权限

### 使用问题
- **文件夹为空**: 检查是否有权限或文件夹确实为空
- **下载失败**: 确认本地存储权限已授予
- **界面卡顿**: 大目录可能需要较长加载时间

### 性能优化建议
- 避免同时展开过多大目录
- 定期清理不需要的书签
- 网络较慢时耐心等待加载
```

# 远程文件浏览器开发文档

## 架构设计

### 核心组件
- **SFTPConnectionManager**: SFTP连接管理
- **ProjectWorkspaceManager**: 项目工作区管理  
- **FileTreeAdapter**: 目录树适配器
- **RemoteFileBrowserAdapter**: 文件列表适配器

### 数据流
1. 用户操作 → Activity事件处理
2. 事件处理 → Manager层业务逻辑
3. 业务逻辑 → SFTP连接操作
4. 异步结果 → UI状态更新

### 状态管理
- 连接状态：通过RxJava Subject管理
- UI状态：通过ViewModel管理（如适用）
- 持久化：SharedPreferences + JSON
